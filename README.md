##Tux of Math Command pour Firefox OS (FR)

###Présentation

Tux de commandement Math est un jeu vidéo conçu pour enseigner les mathématiques aux enfants.

L'objectif de ce projet est de :

- porter cette application pour le Web;
- le rendre utilisable sur un petit écran;
- le rendre utilisable sur un écran tactile.

###Objectifs du jeu

<p align="justify">
Le jeu consiste à trouver la solution d'équation représentée sous forme de comètes avant qu'elle n'atteigne les igloos à protéger pour pouvoir les détruire. Le jeu est divisé en vague de comètes dont la complexité à résoudre et le nombre augmenteront au fur et à mesure.
<br/><br/>
Le joueur a perdu une fois que tous ses igloos sont détruits et chacun d'eux se détruit après avoir reçu trois comètes. Durant le jeu, des comètes spéciales peuvent apparaitre pour donner des bonus spéciaux au joueur (vie, explosion de l'ensemble des comètes, etc...).
</p>

##Tux of Math Command for Firefox OS (EN)

###Presentation

Tux of Math Command is a video game designed to teach mathematics to children.

The objective of this project is to :

- port this application to the web;
- make it usable on a small screen;
- make it usable on a touchscreen.

###Goals of the game

<p align="justify">
The game is to find the solution of equations illustrated as comets before it reaches the igloos protect in order to destroy them. The game is divided into wave of comets whose complexity to solve and the number will increase progressively.
<br/><br/>
The player has lost once all of its igloos are destroyed and each of them is destroyed after receiving three comets. During the game, special comets may appear to give special bonuses to the player (life explosion of all the comets, etc ...).
</p>
